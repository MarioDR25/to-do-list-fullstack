# 📝 To-Do List Fullstack (TS + React + Express)

## 🚀 Tecnologías Utilizadas

### Frontend
* **React 18**
* **TypeScript**
* **Tailwind CSS**

### Backend
* **Node.js** & **Express**
* **Arquitectura MVC**
* **Persistencia en JSON**

## 🛠️ Estructura del Proyecto
```text
to-do-list-fullstack/
├── backend/       # API REST con Express
└── frontend/      # Interfaz de usuario con React
```

## ⚙️ Instalación y Uso

### 1. Clonar el repositorio
```bash
git clone https://github.com/MarioDR25/to-do-list-fullstack.git
cd to-do-list-fullstack
```

### 2. Configurar el Backend
```bash
cd backend
npm install
npm run dev
```

### 3. Configurar el Frontend
```bash
cd ../frontend
npm install
npm run dev
```

## 🧠 Características Principales
* **Operaciones CRUD**: Crear, Leer, Actualizar y Borrar tareas.
* **Persistencia Real**: Los datos se guardan en un archivo local.
* **Tipado Estricto**: Uso de Interfaces para consistencia de datos.
